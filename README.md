### 这是一个mpvue引入原生小程序开发的UI如wux-weapp及iview-weapp的示例


#### 使用说明
``` bash
$ git clone https://github.com/blackjack0v0/mpvue-iview-wux-weapp-demo 
$ cd mpvue-iview-wux-weapp-demo
$ cnpm install
$ cnpm run dev
```
